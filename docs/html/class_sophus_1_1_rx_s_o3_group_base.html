Detailed Description

template<typename Derived>
class Sophus::RxSO3GroupBase< Derived >

RxSO3 base type - implements RxSO3 class but is storage agnostic.

This class implements the group (RxSO3), the direct product of the group of positive scalar matrices (=isomorph to the positive real numbers) and the three-dimensional special orthogonal group SO3. Geometrically, it is the group of rotation and scaling in three dimensions. As a matrix groups, RxSO3 consists of matrices of the form where is an orthognal matrix with and be a positive real number.

Internally, RxSO3 is represented by the group of non-zero quaternions. This is a most compact representation since the degrees of freedom (DoF) of RxSO3 (=4) equals the number of internal parameters (=4).
